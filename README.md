# Spotify Group Session

## Commands 

### Initialization

```bash
npm init
```

### Launch applications
launch to dev(activate server & client)
```bash
npm start
```

launch server
```bash
npm server
```

launch client
```bash
npm client
```

### Install packages

install into an authenication server
```bash
npm i <PACKAGE_NAME>
```

install into React
```bash
cd client
npm i <PACKAGE_NAME>
```
